    How To Use ATR Trailing Stop Tool

    Input Parameters

    Exchange: Name of the exchange used
    Account: Name of the account used for the exchange
    Pair: Cryptocurrency Trading Pair
    Amount: Amount to be used for each order
    Profit: The profit margin between Buy and Sell
    Timer: The time between each new order placed to the exchange
    PostOnly: Order will be placed as limit maker order (available only to exchanges that support this order type)

    Action

    Buy/Sell: The tool will execute a buy and sell cycle
    Sell/Buy: The tool will execute a sell and buy cycle
    Stop: The tool will stop placing new order

    Note

    The tool uses a “follow market” mechanism for order placing.
