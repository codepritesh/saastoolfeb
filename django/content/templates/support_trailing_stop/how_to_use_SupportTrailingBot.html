    How To Use Support Trailing Stop

    Input Parameters

    Exchange: Name of the exchange used
    Account: Name of the account to be used for the exchange
    Pair: Cryptocurrency Trading Pair
    Amount: Size of the order
    Entry Price: Price of the initial order
    Min Profit: The minimum amount of profit the tool should consider
    Trailing Margin: The amount of margin to consider for trailing
    Cancel Threshold (%): The amount of threshold to consider for cancelling the order
    PostOnly: Order will be placed as limit maker order (available only to exchanges that support this order type)
    Gap: (available only when PostOnly is OFF)
    Follow Gap: (available only when PostOnly is OFF)

    Action

    Place Buy: The tool will place a buy order to the exchange. After the order is filled, the tool will monitor the price and trail the order if necessary.

    Place Sell: The tool will place a sell order to the exchange. After the order is filled, the tool will monitor the price and trail the order if necessary.

    Note

    This tool uses a “follow market” mechanism for order placing. Price of the order will be determine based on the prevailing order book prices
