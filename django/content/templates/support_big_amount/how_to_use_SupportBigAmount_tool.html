    How To Use Support Big Amount Bot

    Input Parameters

    Exchange: Name of the exchange used
    Account: Name of the account to be used for the exchange
    Pair: Cryptocurrency Trading Pair
    Price: Price of the order. If price is set to 0, the entry price will be determine based on the current order book
    Amount: Size of the order
    Profit: The amount of profit the tool should consider
    PostOnly: Order will be placed as limit maker order (available only to exchanges that support this order type)
    Gap: (available only when PostOnly is OFF)
    Follow Gap: (available only when PostOnly is OFF)
    Stop loss Value: The amount of stop-loss to be consider for closing the open position
    Place Order ST: Auto places Stop-Loss Order when stop-loss condition is triggered. (ON/OFF)
    Place Order Profit: Auto places Take-Profit Order when open order is filled. (ON/OFF)

    Action

    Place Buy: The tool will place a buy order to the exchange and starts trading.
    Place Sell: The tool will place a sell order to the exchange and starts trading.
